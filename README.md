# 🎉 Event Zen - Event Management System

## 📌 Overview

Event Zen is a full-stack web application developed to manage events efficiently. It allows users to register, log in, create events, and manage event details. The system also includes an admin panel for managing venues and events.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- View Events
- Create Events
- Update/Delete Events

### 🛠 Admin Features
- Admin Login
- Venue Management
- Event Monitoring

---

## 🧑‍💻 Tech Stack
- Frontend: React.js, Bootstrap
- Backend: Node.js, .NET (ASP.NET Core)
- Database: MongoDB
- API Testing: Postman
- Version Control: Git & GitHub
- Containerization: Docker

---

## ⚙️ Installation & Setup
1️⃣ Clone Repository:
git clone https://github.com/soubhagya19/event_gen.git
dc event_gen  
2️⃣ Start Backend (Node):
cd backend 
npm install 
node server.js  
3️⃣ Start Frontend:
cd frontend 
npm install 
npm start  
4️⃣ Start .NET API:
cd VenueAPI 
dotnet run  
---
